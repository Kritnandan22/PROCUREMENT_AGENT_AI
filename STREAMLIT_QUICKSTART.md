## How to Use (3 Easy Steps)

### Step 1️⃣: Select Organization
- Choose **"Use Default"** (recommended)
- OR pick your specific organization ID

### Step 2️⃣: Select Workflow
Pick what you want to analyze:
- 🚨 **exception-triage** - Find issues (late POs, shortages, excess)
- ⏰ **late-supplier** - Which suppliers are late?
- 📦 **safety-stock** - Is inventory OK?
- 💰 **price-anomaly** - Any weird prices?
- 🔄 **demand-to-po** - Convert demand to orders
- 📊 **spend-analytics** - See spending patterns

### Step 3️⃣: Click RUN & Download
1. Click **"▶️ RUN ANALYSIS"** (blue button)
2. Wait 2-5 seconds
3. Click **"⬇️ Download Excel Report"**
4. Excel file saves to your Downloads folder!
